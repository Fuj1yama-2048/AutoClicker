# 更新日志

## v1.1.1 (2026-05-30)

### 新增功能

- **热键自定义** — 设置窗口内可修改开始/录制/回放的全局热键，支持组合键（Ctrl/Shift/Alt/Win）
- **设置持久化** — 所有设置项（置顶、随机间隔、位置微抖、采样率、关闭行为、开始时隐藏、录制自窗口光标）保存到注册表，重启软件保持上次配置
- **默认文件路径** — 设置窗口内可指定导入/导出的默认文件夹，支持手动输入或浏览选择
- **只读展示框（CreateDisplay）** — 新增辅助控件，居中文本展示，不可编辑不可选中
- **注册表清除脚本** — `clear_reg.bat` 一键清除所有注册表配置，恢复首次使用状态
- **版本号显示** — 标题栏显示版本号（连点器 Auto Clicker v1.1.1）

### 修复

- **热键按钮动态显示** — 所有热键按钮文字不再写死 F6/F7/F8，跟随实际配置动态更新
- **底部状态栏颜色统一** — 修复动态文字和静态文字背景色不一致的问题
- **固定位置连点速度** — 移除 ClickAtFixedPos 中的 Sleep(10)，高频连点不再变慢
- **设置窗口文字遮盖** — 设置窗口改为独立窗口，不再导致主窗口底部标签文字被刷掉
- **初次启动热键未加载** — 修复程序启动时热键标签显示默认值而非注册表保存值的问题
- **导入/导出默认路径** — 设置了默认路径后，文件对话框自动定位到指定目录
- **程序图标** — 修复构建脚本缺少资源编译导致 exe 无图标的问题
- **代码可读性** — 控件 ID 和辅助函数添加中文注释，竖向排版对齐
- **Build 脚本路径修正** — 修复 build_msvc.bat 中的错误路径
